# 尽调小助手

律师尽职调查全流程工具，支持批量解析PDF/Word尽调文件，提取关键信息生成项目表和尽调报告。

## 功能特性

- 📁 **批量文件解析**：自动识别嵌套文件夹，读取决议文件、投资合同、合伙协议等
- 📊 **项目表生成**：提取关键信息整理成Excel表格
- 📝 **尽调报告生成**：基于模板生成Word格式报告
- 📋 **待补充材料清单**：自动识别缺失材料
- 🔄 **迭代评估**：支持补充材料后重新评估

## 使用流程

1. 运行 `/尽调小助手`
2. 上传文件或选择跳过
3. 填写公司信息（可跳过）
4. 补充特殊情况备注（可跳过）
5. 选择报告章节（默认全部）
6. 选择报告模板
7. 生成报告

## 项目结构

```
尽调小助手/
├── .claude/skills/       # Skill配置
├── tools/                # Python工具脚本
├── templates/            # 模板文件
├── input/                # 待处理文件
├── output/               # 生成的文件
└── requirements.txt      # Python依赖
```

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 准备文件

将待处理的文件夹或压缩包放入 `input/` 目录。

### 3. 运行

在Trae中输入：
```
/尽调小助手
```

## 技术栈

- Python 3.8+
- PyPDF2 / PyMuPDF - PDF解析
- python-docx - Word文档处理
- openpyxl - Excel表格生成

## 约束与注意事项

1. 不得编造不存在的数据和信息
2. 缺乏数据时标注[待补充]
3. 每一步暂停让用户确认
4. 预留飞书等平台集成接口
