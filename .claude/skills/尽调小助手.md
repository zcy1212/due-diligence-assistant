# 尽调小助手

法律尽职调查全流程工具，支持批量解析PDF/Word尽调文件并提取关键信息生成项目表，AI评估生成尽调报告与待补充材料清单，支持迭代补充和飞书在线协作。

## 功能说明

### 主要功能
1. **批量文件解析**：上传文件夹或压缩包，自动识别嵌套的项目文件夹，读取大量决议文件、投资合同、合伙协议等
2. **项目表生成**：提取文件关键信息并整理成Excel表格
3. **尽调报告生成**：基于项目表和用户填写信息生成Word格式尽调报告
4. **待补充材料清单**：自动识别缺失材料并生成清单
5. **迭代评估**：支持补充材料后重新评估生成新报告

### 特性
- 每一步暂停让用户选择确认
- 所有用户填写步骤可跳过，空缺信息标注[待补充]
- 支持用户自定义填写选项
- 预留飞书等平台集成接口

## 项目文件说明

```
trea dd/
├── CLAUDE.md                          # 项目规范
├── .claude/
│   └── skills/
│       └── 尽调小助手.md             # 本文件
├── tools/
│   ├── file_reader.py                 # 文件读取和解析
│   ├── excel_writer.py                # Excel表格生成
│   └── report_writer.py               # Word报告生成
├── templates/
│   ├── 报告模板.txt                   # 默认尽调报告模板
│   └── 项目表示例.xlsx                # 项目表格式示例
├── input/                             # 用户待处理的文件夹或压缩包
├── output/                            # 生成的文件输出目录
│   ├── 项目表.xlsx
│   ├── 尽调报告.docx
│   └── 待补充材料清单.docx
└── temp/                              # 临时文件（自动清理）
```
