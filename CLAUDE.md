# 尽调小助手项目规范

## 项目说明
律师尽职调查全流程工具，支持批量解析PDF/Word尽调文件并提取关键信息生成项目表，AI评估生成尽调报告与待补充材料清单，支持迭代补充和飞书在线协作

## 目录结构
```
trea dd/
├── CLAUDE.md                          # 项目规范
├── .claude/
│   └── skills/
│       └── 尽调小助手.md             # 主Skill配置
├── tools/
│   ├── file_reader.py                 # 文件读取和解析
│   ├── excel_writer.py                # Excel表格生成
│   └── report_writer.py               # Word报告生成
├── templates/
│   ├── 报告模板.txt                   # 默认尽调报告模板
│   └── 项目表示例.xlsx                # 项目表格式示例
├── input/                             # 用户待处理的文件夹或压缩包
├── output/                            # 生成的文件输出目录
│   ├── 项目表.xlsx
│   ├── 尽调报告.docx
│   └── 待补充材料清单.docx
└── temp/                              # 临时文件（自动清理）
```

## 工作流程
1. 用户上传文件到 input/ 目录
2. 运行 /尽调小助手 开始流程
3. 每步操作暂停，提供选项供用户选择，支持自定义填写
4. 所有填写步骤可跳过，空缺信息标注[待补充]
5. 支持迭代补充材料后重新评估

## 约束
- 不得编造不存在的数据和信息
- 缺乏数据或信息时标注[待补充]
- 密钥、token、密码不进代码、不进commit、不进日志
